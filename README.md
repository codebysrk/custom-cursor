# Custom Cursor

A simple vanilla JavaScript project that replaces the default system cursor with a custom image cursor.

## Features

- **Custom Cursor**: Uses a PNG image (`cursor.png`) as a cursor.
- **Video Background**: Displays a full-screen background video (`house.mp4`).
- **Smooth Interaction**: Updates cursor position in real-time using JavaScript `mousemove` event.

## Project Structure

- `index.html`: Structure of the page, including the video and script.
- `style.css`: Styling for the full-screen layout and the custom cursor appearance.
- `script.js`: Logic to track mouse movement and update the custom cursor's position.

## How to Run

1. Clone the repository.
2. Open `index.html` in your web browser.

## Technologies

- HTML5
- CSS3
- JavaScript
